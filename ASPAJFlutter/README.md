# 🚀 4LLASET Mobile (Sistem Peminjaman Aset Jurusan)
**Premium Asset Management Edition**

4LLASET Mobile adalah aplikasi frontend berbasis Flutter yang dirancang untuk memudahkan siswa dan petugas dalam mengelola peminjaman aset di lingkungan sekolah secara digital, cepat, dan transparan.

---

## ✨ Fitur Utama
*   **Multi-Role Access**: Mendukung peran Siswa, Petugas Jurusan (Officer), dan Admin.
*   **Katalog Aset Modern**: Tampilan aset berbasis grid dengan kategori jurusan dan status stok real-time.
*   **Sistem Keranjang & Checkout**: Proses peminjaman yang intuitif seperti aplikasi e-commerce.
*   **Monitoring Peminjaman**: Pelacakan status peminjaman dari tahap pengajuan hingga pengembalian.
*   **Premium UI/UX**: Desain modern menggunakan Glassmorphism, Google Fonts, dan animasi halus.

---

## 🛠️ Persiapan Pengembangan
1.  **Clone Repository**:
    ```bash
    git clone [url-repository]
    ```
2.  **Konfigurasi API**:
    Edit file `lib/services/api_service.dart` dan arahkan `BASE_URL` ke alamat server Laravel Anda.
3.  **Install Dependencies**:
    ```bash
    flutter pub get
    ```
4.  **Jalankan Aplikasi**:
    ```bash
    flutter run
    ```
